~New functions added in this version~

Новый функционал, добавленный с 01 мая 2025 года:

1) Добавлен результат в % - целое число от 0 до 100 без округления, которое получается путем деления количества правильных ответов на общее количество слов
2) Если результат менее 100%, то бот выводит: "Вы почти у цели!" (ранее выводилось поздравление для любого результата - теперь только если 100%)

### Обновление уровня клиента (level_id_current)

Скрипт: `db/migrations/update_client_levels.py`

Функции:
- Расчёт фактического уровня клиента (A1–C2) на основе завершённых викторин
- Проверка выполнения порогов по coverage (>= 80%) и accuracy (>= 80%) для уровня
- Обновление поля `client_analytics.level_id_current` по расписанию

Запуск:
- вручную: `python db/migrations/update_client_levels.py`
- или из `cronjobs/run_daily_updates.sh`
